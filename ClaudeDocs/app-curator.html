Apps · building datasets

Curator

A Dataset is a hand-picked set of epochs for one analysis. Curator is where you make it: load a cell, step through its epochs, drop the bad ones, and save what’s left under a name. You usually reach it from DataGrouper’s To Curator button at the Cell level.

The workflow

Load the cell

Confirm the Working cell is the one you meant (DataGrouper delays can occasionally hand over the wrong one), then Load cell.

Pick the epoch group

Use Quick select epoch group to choose the protocol you want — spots multi size, moving bar, and so on.

Step through the epochs

The Epoch number slider (or the Current epoch field) moves through trials; the raw trace draws up top. Use Data channel and X axis to change what’s plotted. Watch for noise, lost seals, or perfusion artifacts.

Filter to the epochs you want

Build a Filter pattern string from the numbered parameters in the filter table. @n refers to parameter n, && is AND, and &! is AND-NOT. For example @1 && @2 && @3 && @4 && @5 keeps everything matching all five; swapping the last to &! @5 keeps everything except those. Click Run filter to apply — kept epochs turn green.

Save the dataset

Save dataset prompts for a name and writes the selected epochs as a new sln_symphony.Dataset for this cell. Rename dataset and Delete dataset manage existing ones.

Reading the tables Parameters for selected epoch shows the values for the one epoch you’re on; Parameters with multiple values shows which parameters vary across the group — those are usually what you filter on.
