GUI apps

The apps you’ll actually click

Most day-to-day work with the database happens through MATLAB App Designer GUIs — no SQL, no query code. You launch one by typing its name in the MATLAB Command Window (e.g. DataGrouper_V2) and pressing Enter. These pages walk through the main ones.

Before any app will work You need DataJoint set up in MATLAB, a VPN or on-site network connection, the FSMResfiles share mounted, and an up-to-date clone of DJ_schwartzlab on your MATLAB path. Apps can be slow — give them a moment after launch and don’t click too fast.
Find · curate · analyze

DataGrouper V2

The hub. Search the database, group epochs into datasets, add and create tags, assign cell types, run analyses, plot results, and export — all in one window.

 Re-run saved queries

QueryBrowser

Browse every query the lab has saved, filter by person or project, and re-run one to pull its datasets or analysis results straight into MATLAB.

 Build datasets

Curator

Step through a cell’s epochs, filter out the bad or unwanted ones, and save a clean set as a named Dataset.

 Detect spikes

Spike Detector

Turn raw traces into spike times: threshold, review, hand-correct, and push the result to the SpikeTrain table.

 Calcium imaging · ROIs

ROI Tool

Draw or auto-detect regions of interest on a functional-imaging run, pull their fluorescence traces, and write the ROIs to the database.

 Part of DataGrouper

Plot Designer

Lay out a reusable grid of plot panels that DataGrouper can arrange results into.

 Recording timeline

Diary Plotter

See how a current-clamp cell’s per-epoch stats drift across a session, mark condition windows, and compare two stretches waveform-by-waveform.

How they fit together

A typical path from raw recording to a plotted result touches several of these in turn:

Group epochs into a dataset

In DataGrouper you find a cell, then send it To Curator to pick the good epochs and save a Dataset.

Detect its spikes

For spiking data, To Spike Detector turns the raw traces into SpikeTrain rows.

Tag, analyze, and explore

Back in DataGrouper, add tags, run an analysis on the datasets, and plot the results.

Save the query for later

Save the query so you (or a labmate) can re-run it anytime from QueryBrowser.

Other apps you may run into — CellTyper, QueryBuilder, DataExplorer, PlotDesigner, H5_Exporter, the AnimalInterface, and the imaging tools — follow the same launch-by-name pattern.
