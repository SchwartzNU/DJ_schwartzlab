Apps · part of DataGrouper

Plot Designer

When you want several plots arranged together — a grid of panels rather than one figure at a time — Plot Designer is where you lay that out once and reuse it. It’s a small helper that opens from DataGrouper’s Plots panel via the Plot Designer button.

What it makes

A layout — a named grid that says “this plotter here, that one there.” Once saved, it shows up in DataGrouper’s Layout dropdown (next to the save-format buttons), so Save all plots in folder arranges the figures the way you designed instead of one per file.

The workflow

Open it

In DataGrouper’s Plots panel, click Plot Designer. DataGrouper waits for you to finish, then refreshes its Layout list automatically.

Set the grid size

Enter Rows and Columns. The grid rebuilds to that shape with equal-sized cells (changing either number resets the grid).

Assign a plotter to each cell

Every cell is a dropdown listing your available plotters — the plotter functions in your plotters folder, the same ones offered by DataGrouper’s Plotter menu. Pick one per cell; leave a cell on Select a plot to keep it empty.

Save the layout

Save writes the layout to a .mat file (in the figure-layouts folder) under a name you choose. Back in DataGrouper it appears in the Layout dropdown, ready to reuse on any result.

Where plotters come from The choices in each cell are simply the .m plotter files on your plotters path. Add a new plotter there and it becomes available both here and in DataGrouper’s Plotter menu.
